Definiones Backend

Backend

    Parte de una aplicacion web que maneja la logica del servidor y la base de datos.

API

    Interfaz que permite la comunicacion entre diferentes sistemas.

Base de Datos

    Sistemas donde se almacenan y gestionan los datos de una aplicacion.

Framawork Backend

    Conjunto de herramientas y librerias para desarrollar servidores y aplicaciones web.

Servidor

    Computadora o software que maneja peticiones y entrega datos a los clientes.
